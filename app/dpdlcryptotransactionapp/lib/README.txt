
To run the Dpdl example you need to download and compile the C library used.

The 'libethc' C library used in this example is available in the following repository:

https://github.com/mhw0/libethc.git

To run the Dpdl example, clone the above repository, compile the 'libethc' library and place it in
this folder (with the following naming: libethc.so for Linux/Unix, libethc.dynlib for Mac OS, libethc.dll on Windows)
